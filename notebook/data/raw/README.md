### Required Files

- **train.csv** – Superstore Sales Datasethttps://www.kaggle.com/datasets/rohitsahoo/sales-forecasting
- **vgsales.csv** – Video Game Sales Dataset
  https://www.kaggle.com/datasets/gregorut/videogamesales

After placing both files in the `data/raw/` directory, run the notebook to generate the processed datasets, trained models, forecasts, and reports required by the Streamlit dashboard.
